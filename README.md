# Smart-Visit Festival d'Avignon

Projet Big Data M2 - 2025/2026.

## Stack technique
- **ETL** : Apache Spark (PySpark)
- **Stockage** : MongoDB (données documentaires) + Neo4j (graphe de proximité)
- **Backend** : FastAPI (Python)
- **Mobile** : Flutter / React Native
- **Orchestration** : Docker Compose

## Lancer l'infrastructure

docker-compose up -d
